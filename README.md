## 📝 简介

AstrBot 广告检测插件是一款专为 QQ 群设计的智能广告检测工具，支持正则、AI等多种检测方式，可自动处理违规广告内容，有效维护群聊秩序。

## ✨ 功能特性

### 🎯 检测能力
- **正则检测** - 自定义关键词正则匹配，灵活配置检测规则
- **AI 检测** - 调用 LLM 智能识别文本和图片广告内容
- **引用消息检测** - 同时检测被引用的消息内容

### ⚙️ 处理动作
- 自动撤回违规消息
- 发送警告通知提醒用户
- 达到阈值自动踢出用户

### 🛡️ 管理功能
- 用户白名单 - 特定用户不受检测
- 群组白名单 - 整个群组不受检测
- 违规记录管理 - 查看和重置用户违规次数
- 丰富的管理指令 - 便捷的群管理工具

## 🚀 快速开始

### 安装
1. 下载本插件到本地
2. 将插件放入 AstrBot 的 `data/plugins` 目录
3. 重启 AstrBot 自动安装依赖
4. 在 WebUI 中配置插件参数

### 配置说明

#### 基本配置
在 AstrBot WebUI 中配置以下选项：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用正则检测 | 使用正则表达式检测广告 | ✅ |
| 启用AI检测 | 使用 AI 检测文本和图片广告 | ❌ |
| 启用图片检测 | 检测图片中的广告内容 | ✅ |
| 启用引用消息检测 | 检测引用的消息 | ❌ |
| 启用撤回 | 自动撤回违规消息 | ✅ |
| 启用警告 | 发送警告消息 | ✅ |
| 启用踢出 | 达到阈值后踢出 | ❌ |
| 警告阈值 | 违规次数阈值 | 3 |

## 📚 管理命令

所有命令均需群管理员权限。

| 命令 | 说明 |
|------|------|
| `/ad_violation <用户ID>` | 查看用户违规记录 |
| `/ad_reset_violation <用户ID>` | 重置用户违规记录 |
| `/ad_whitelist_user <add/remove> <用户ID>` | 添加/移除用户白名单 |
| `/ad_whitelist_group <add/remove>` | 添加/移除群组白名单 |
| `/ad_help` | 显示帮助信息 |
