# AstrBot 广告检测插件

基于 AstrBot 的 QQ 群广告检测插件，支持文本正则检测、文本 AI 检测、图片 OCR 检测等多种检测方式。

## 功能特性

### 检测能力
- **正则检测：自定义关键词正则匹配
- **文本 AI 检测：调用 LLM 智能识别文本广告
- **图片 OCR 检测：使用 EasyOCR 提取图片文字后检测
- **引用消息检测：同时检测引用的消息

### 处理动作
- 自动撤回违规消息
- 发送警告通知
- 达到阈值自动踢出用户

### 管理功能
- 用户白名单
- 群组白名单
- 违规记录管理
- 丰富的管理指令

## 快速开始

### 安装
1. 将插件放入 AstrBot 的 `data/plugins` 目录
2. 重启 AstrBot 自动安装依赖
3. 在 WebUI 中配置插件

### 配置说明

#### 基本配置
在 AstrBot WebUI 中配置以下选项：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用正则检测 | 使用正则表达式检测广告 | 是 |
| 启用文本AI检测 | 使用 AI 检测文本广告 | 否 |
| 启用图片检测 | 检测图片中的广告内容 | 是 |
| 图片检测模式 | ocr_only 或 ocr_ai | ocr_only |
| 启用引用消息检测 | 检测引用的消息 | 否 |
| 启用撤回 | 自动撤回违规消息 | 是 |
| 启用警告 | 发送警告消息 | 是 |
| 启用踢出 | 达到阈值后踢出 | 否 |
| 警告阈值 | 违规次数阈值 | 3 |

## 管理命令

所有命令均需群管理员权限。

| 命令 | 说明 |
|------|------|
| `/ad_violation <用户ID>` | 查看用户违规记录 |
| `/ad_reset_violation <用户ID>` | 重置用户违规记录 |
| `/ad_whitelist_user <add/remove> <用户ID> | 添加/移除用户白名单 |
| `/ad_whitelist_group <add/remove>` | 添加/移除群组白名单 |
| `/ad_help` | 显示帮助信息 |
