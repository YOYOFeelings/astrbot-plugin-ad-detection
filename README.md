# astrbot-plugin-ad-detection

Astrbot QQ群广告检测插件

## 功能特性

### 检测功能
- 正则表达式检测 - 使用关键词匹配广告
- 文本AI检测 - 使用 LLM 智能识别广告
- 图片AI检测 - 预留接口（待完善）
- 引用消息检测 - 检测被引用的消息

### 处理功能
- 自动撤回违规消息
- 发送警告消息
- 达到阈值自动踢出用户

### 管理功能
- 用户白名单 - 白名单内用户不受检测
- 群组白名单 - 白名单内群组不受检测
- 违规记录管理 - 查看和重置用户违规次数
- 管理员命令 - 丰富的管理指令

### 配置特性
- 所有功能独立开关
- 文本AI和图片AI使用独立的模型提供商配置
- 自定义正则规则
- 自定义警告消息和阈值

## 安装

1. 下载本插件
2. 将插件放入 AstrBot 的 `data/plugins` 目录
3. 重启 AstrBot

## 配置

在 AstrBot 的 WebUI 配置界面中配置插件：
- 启用/禁用各项功能
- 选择 AI 模型提供商（文本检测和图片检测独立配置）
- 设置正则检测规则
- 配置警告阈值和消息

## 管理员命令

| 命令 | 说明 |
|------|------|
| `/ad_violation <用户ID>` | 查看用户违规记录 |
| `/ad_reset_violation <用户ID>` | 重置用户违规记录 |
| `/ad_whitelist_user <add/remove> <用户ID>` | 管理用户白名单 |
| `/ad_whitelist_group <add/remove>` | 管理群组白名单 |
| `/ad_help` | 显示帮助信息 |

## 注意事项

- 仅群管理员可执行管理命令
- 数据库存储在 AstrBot 的 data 目录下
- 图片检测功能待完善
